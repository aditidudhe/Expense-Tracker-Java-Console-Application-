# Expense Tracker (Java Console Application)

This is a Java-based console application that allows users to track and manage their expenses. It supports tracking daily, weekly, monthly, and yearly expenses, calculating savings, and suggesting investment opportunities based on savings.

## Features

- Add daily expenses with date, time, description, and amount
- Set a minimum balance
- Set and update monthly income
- View expenses:
  - Daily
  - Weekly
  - Monthly
  - Yearly
  - For a specific month and year
- Delete expenses by description
- View monthly and yearly savings
- Suggest investment opportunities based on current savings
- Console-based menu for easy interaction

## Technologies Used

- Java (JDK 8 or above)
- ArrayList
- Java Time API (LocalDate and LocalTime)
- Scanner for input
